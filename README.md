# rust-hangman

CLI Hangman game written in pure Rust for fun.

## Installation

1. **Clone the repository:**
```bash
git clone https://github.com/Egzavyer/rust-hangman.git
```
2. **Build the project:**
```bash
cargo build --release
./target/release/hangman
```
or
```bash
cargo run
```
